# HEARTBEAT.md - QLD Deals Posting Schedule

## Posting Times (Brisbane/AEST)
- 🌅 Morning deals: 6:30am
- 🍱 Lunch deals: 12:00pm  
- 🌆 Afternoon/Evening deals: 5:00pm

## Post Template (Dark Theme)

Every post should follow this format:

1. **🖼️ Hero Image** - Large, attention-grabbing image relevant to the deals
2. **📢 Bold Headline** - e.g., "Bunnings Half Price DIY Deals!"
3. **🏷️ Keywords with Icons** - wrench 🔧, sprout 🌱, house 🏠, etc.
4. **🔥 Discount** - "Up to 50% OFF!" prominently displayed
5. **📝 Deal Details** - Product, original price → sale price
6. **❓ Engagement Question** - Drive comments
7. **#️⃣ Hashtags** - #QLDDeals #QueenslandDeals #SaveMoneyAU

## Content Categories by Time
- **Morning (6:30am):** Groceries, breakfast, snacks
- **Lunch (12pm):** Ready meals, sushi, salads, drinks
- **Afternoon (5pm):** Home & DIY, health & beauty, weekend fun

## Content Rules
- ONLY real, verified deals (no fake prices)
- Mix of: deal posts, engagement posts, local Hervey Bay content
- End every post with a question to drive comments
- Always include relevant hashtags

## Posting Workflow
1. Search for current Woolworths/Coles/Bunnings/Chemist Warehouse/ALDI half price specials
2. Create 3-5 ready-to-post Facebook posts using the template
3. Add Unsplash image for each post
4. Post to QLD Deals Facebook page (Page ID: 1107085552479301)
5. Save content to workspace as Daily_Deals_YYYY-MM-DD.md

## Priority
- Real deals over quantity
- If no good deals found, post engagement/helper content instead
- Keep it honest — say "not worth it" when true
